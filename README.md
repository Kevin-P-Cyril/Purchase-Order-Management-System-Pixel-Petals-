PixelPetals is a web-based Inventory and Purchase Order Management System (POMS) developed to simplify semiconductor procurement and inventory operations. It provides a modern and user-friendly dashboard with role-based access for Admin and Users, enabling efficient product management, purchase order creation, and status tracking.

🚀 Features
🔐 Authentication
Separate login for Admin and User
Basic validation for secure access
📦 Inventory Management
Add new products
Update existing products
View product list
Visualize inventory using charts
🧾 Purchase Order Management (POMS)
Multi-step purchase order creation:
Basic Information
Parts Details
Terms
Review
Includes supplier, buyer, and semiconductor details
🔄 PO Status Tracking
Admin can update purchase order status (Draft, Approved, Cancelled, etc.)
Users can check purchase order status dynamically
📊 Dashboard
Interactive charts using Chart.js
Recent activity tracking
Clean and responsive UI
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
UI Framework: Tailwind CSS
Charts: Chart.js
Icons: Font Awesome, Lucide
